type Kind = Type

The key type representing kinds in the compiler. Invariant: a kind is always in one of these forms: 

 FunTy k1 k2
 TyConApp PrimTyCon [...]
 TyVar kv   -- (during inference only)
 ForAll ... -- (for top-level coercions)
