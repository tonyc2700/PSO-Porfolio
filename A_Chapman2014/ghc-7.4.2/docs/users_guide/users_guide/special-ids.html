7.18. Special built-in functions

GHC has a few built-in functions with special behaviour. These are now described in the module GHC.Prim in the library documentation. In particular:

  • inline allows control over inlining on a per-call-site basis.

  • lazy restrains the strictness analyser.

  • unsafeCoerce# allows you to fool the type checker.
